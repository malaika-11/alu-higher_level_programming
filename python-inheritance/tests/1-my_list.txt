===========================
Tests for MyList Class
===========================

First, import MyList from the module:
>>> MyList = __import__('1-my_list').MyList

Check basic sorting:
>>> my_list = MyList([1, 4, 2, 3, 5])
>>> print(my_list)
[1, 4, 2, 3, 5]
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]
>>> print(my_list)
[1, 4, 2, 3, 5]

Check with an empty list:
>>> my_list = MyList()
>>> my_list.print_sorted()
[]

Check with negative numbers:
>>> my_list = MyList([10, -5, 0, 3])
>>> my_list.print_sorted()
[-5, 0, 3, 10]

Check with duplicate values:
>>> my_list = MyList([1, 2, 2, 1])
>>> my_list.print_sorted()
[1, 1, 2, 2]
