The MyList class
================

Import the class:
>>> MyList = __import__('1-my_list').MyList

Check that append works correctly:
>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)
>>> print(my_list)
[1, 4, 2, 3, 5]

Verify that print_sorted() works after appending:
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]

Verify the original list is still the same:
>>> print(my_list)
[1, 4, 2, 3, 5]

Check appending negative numbers:
>>> my_list.append(-5)
>>> print(my_list)
[1, 4, 2, 3, 5, -5]
>>> my_list.print_sorted()
[-5, 1, 2, 3, 4, 5]
