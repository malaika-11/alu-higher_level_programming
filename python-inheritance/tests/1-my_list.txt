The MyList class
================

Import MyList:
>>> MyList = __import__('1-my_list').MyList

Check that append works and the list prints correctly:
>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)
>>> print(my_list)
[1, 4, 2, 3, 5]

Check that print_sorted() works:
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]

Check that the original list is still unsorted:
>>> print(my_list)
[1, 4, 2, 3, 5]

Check with an empty list:
>>> my_list = MyList()
>>> my_list.print_sorted()
[]

Check with negative numbers:
>>> my_list = MyList([10, -5, 0])
>>> my_list.append(-100)
>>> my_list.print_sorted()
[-100, -5, 0, 10]
