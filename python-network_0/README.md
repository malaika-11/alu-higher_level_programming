# Python - Network #0

This project contains Bash scripts that practice HTTP requests using curl.

## Tasks

- 0-body_size.sh: Display response body size in bytes
- 1-body.sh: Display response body for 200 status code
- 2-delete.sh: Send DELETE request and display response
- 3-methods.sh: Display allowed HTTP methods
- 4-header.sh: Send GET request with custom header
- 5-post_params.sh: Send POST request with parameters
