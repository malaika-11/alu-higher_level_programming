# Python - More Classes and Objects

## Description
This project explores Object-Oriented Programming in Python by building a Rectangle class step by step, adding validation, methods, and special behaviors.

## Key Concepts
- Classes vs instances  
- Attributes (public, private)  
- Methods and `self`  
- `__init__`, `__str__`, `__repr__`, `__del__`  
- Properties (getters/setters)  
- Class vs instance attributes  
- Class methods & static methods
